# Tournament Prediction Maker

A web-based tool for creating and predicting tournament brackets. Supports both Single and Double Elimination formats with a dark-themed, responsive interface.

## Features

*   **Formats**: Single Elimination and Double Elimination.
*   **Dynamic Bracket Generation**: Automatically handles "Byes" and bracket sizing based on the number of teams.
*   **Interactive Predictions**: Click teams to advance them to the next round.
*   **Drag and Drop**:
    *   Drag teams from the sidebar to the bracket.
    *   Swap teams within the bracket.
    *   Swap entire matches for organization.
*   **Score Tracking**: Input scores for each match.
*   **Custom Labels**: Rename matches (e.g., "Grand Finals").
*   **Fit to Screen**: Automatically scales the bracket to fit your viewport.

## How to Use

1.  **Setup**:
    *   Enter a Tournament Name.
    *   Enter Team Names (one per line) in the text area.
    *   Select the Format (Single or Double Elimination).
    *   Click **Create Bracket**.

2.  **Making Predictions**:
    *   **Advance**: Click on a team name to advance them to the next round.
    *   **Scores**: Enter scores in the small input boxes next to team names.
    *   **Edit**: Drag and drop teams or matches to rearrange the bracket if needed.

3.  **Controls**:
    *   **Edit Teams**: Go back to the setup screen.
    *   **Reset Predictions**: Clears all winners and scores, resetting the bracket to the initial matchups.
    *   **Fit to Screen**: Zooms the bracket to fit within your current window size.
